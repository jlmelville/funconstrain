# Test Function Data for use with Optimx

This function provides formatted data about each of the functions in
this package for ease of use with the `optimx` package.

## Usage

``` r
fufn(fnum)
```

## Arguments

- fnum:

  Function number (1-35) to return data for.

## See also

[`fufnrun`](https://jlmelville.github.io/funconstrain/reference/fufnrun.md)
to run optimx using this data.
